# HR Årshjul

Ett verktyg för att snabbt skapa visuella HR-årshjul för olika kunder.

## Vad det gör

- Lägg till och ta bort **ringar** efter kundens behov (Arbetsmiljö, Utveckling, Lön & förmåner, Kompetensutveckling, ...)
- Placera **aktiviteter** med startvecka och längd — hjulet uppdateras live
- **Ladda ner** hjulet som en högupplöst PNG för att dela med kunden
- **Ladda upp** samma bild senare och fortsätt redigera (projektdatan ligger inbäddad i bilden)

## Teknik

Vanlig HTML, CSS och JavaScript — inget byggsteg, inga ramverk. Öppna `index.html` i en webbläsare så funkar allt.
